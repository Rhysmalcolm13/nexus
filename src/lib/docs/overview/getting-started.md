# Getting Started

## Prerequisites

- Node.js 16+
- PostgreSQL
- pnpm/npm/yarn

## Installation

1. Clone the repository
2. Install dependencies: `pnpm install`
3. Set up environment variables
4. Initialize database: `pnpm prisma migrate dev`
5. Start development server: `pnpm dev`
